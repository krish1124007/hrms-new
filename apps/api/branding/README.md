# Branding Assets

Drop your company logo here. The payslip generator looks for these files:

- `company-logo.png` — embedded in the payslip PDF header (max 200×60 px renders at ~50px tall)
- `company-logo-large.png` (optional) — used in any future ID-card / certificate templates

Recommended:
- PNG with transparent background
- Wide, short aspect ratio (≈ 4:1) — fits the header band cleanly
- 600px wide @ 150 DPI = ~3 inches

If no logo is present, the PDF falls back to the company name in plain text.
