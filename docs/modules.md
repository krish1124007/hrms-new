# Module Reference

DD HRMS includes **56 modules** organized into **11 categories**. Module availability depends on the subscription plan.

**Plan Legend:**
- S = Starter (INR 199/user/month)
- P = Professional (INR 399/user/month)
- E = Enterprise (INR 799/user/month)
